# Hotel Entrance Background Image

## Missing Image File

The kiosk is looking for: `Hotel_entrance.png` in this folder.

## To Add the Image:

1. Find or create a hotel entrance/lobby image
2. Save it as `Hotel_entrance.png` 
3. Place it in this `/public` folder
4. Refresh the kiosk window

## Recommended Image Specs:
- **Size**: 1920x1080 or higher
- **Format**: PNG or JPG (rename to .png)
- **Content**: Hotel lobby, reception desk, entrance doors

## Current Fallback:
Until you add the image, the kiosk shows a blue gradient background.

## Quick Test:
You can use any hotel lobby image from the internet, just rename it to `Hotel_entrance.png` and place it here.
